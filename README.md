# Rhombix Task 3

A Flutter-based internship task application built for Rhombix Technologies. This app demonstrates proficiency in user authentication, geolocation services, interactive maps, multi-language support, and payment processing -- all within a single cohesive mobile application.

## Features

- **User Authentication** -- Sign-up and sign-in flows for secure access
- **Geolocation & Reverse Geocoding** -- Detects user location and converts coordinates into readable addresses
- **Interactive Map Integration** -- Displays locations on a map using flutter_map
- **Multi-Language Support** -- Fully localized in English, Spanish, and Urdu via easy_localization
- **Stripe Payment Integration** -- Complete checkout flow powered by Stripe

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | Flutter |
| Maps | flutter_map |
| Location | geolocator |
| Payments | Stripe |
| Localization | easy_localization |

## Getting Started

### Prerequisites

- Flutter SDK installed ([installation guide](https://docs.flutter.dev/get-started/install))
- A Stripe account with API keys configured
- An emulator or physical device for testing

### Installation

```bash
git clone https://github.com/Faze789/rhombix_task_3.git
cd rhombix_task_3
flutter pub get
flutter run
```
